### 文章、教程与话题

[Raft in Python](https://chelseatroy.com/wp-content/uploads/2020/09/Chelsea_Troy_Raft_In_Python.pdf)
`Raft` 算法是一种分布式系统共识算法，其设计者设计（并大力推销）的目的是为了比它的前辈 `Paxos` 算法更容易理解和实现。逐 步了解如何用 `Python` 实现 `Raft`。

[The NEW Match-Case Statement in Python 3.10](https://www.youtube.com/watch?v=2qJavL-VX9Y) ![img](https://mcusercontent.com/e2e180baf855ac797ef407fc7/images/af76283a-6e65-436c-967a-900427cf6399.png)
`Python 3.10` 开始充满了许多迷人的新特性。其中有一个特别引起了我的注意--结构模式匹配--或者我们大多数人都知道的 `switch/case` 语句。让我们来看看它是如何工作的。

[What Microsoft knows about you and how you can use it to generate your own insights](https://t.co/3Rl6YHSJlq)
如何使用微软的 `Graph API` 和 `Django` 构建自己的 `Dashboard App`，分析来自 `OneDrive、Outlook` 等数据的完整指南。

[如何在Pytest中使用Selenium进行参数化操作](https://www.lambdatest.com/blog/parameterization-in-pytest-with-selenium/)
在本篇 `Selenium Python` 教程中，我们将看到 `pytest.fixture、@pytest.mark.parametrize decorator` 和 `pytest_generate_tests()` 如何用于 `pytest` 中的参数化。

[How To Make a Photo Trap With a Raspberry Pi and a Sony Alpha](https://petapixel.com/2021/03/24/how-to-make-a-photo-trap-with-a-raspberry-pi-and-a-sony-alpha/)
You can make your own squirrel-activated remote camera with a Sony Alpha, a Raspberry Pi, and these simple instructions.

[使用Python和Terraform将AWS日志洞察力作为CloudWatch的指标](https://www.thedigitalcatonline.com/blog/2021/03/22/aws-log-insights-as-cloudwatch-metrics-with-python-and-terraform/)
逐步指导如何使用 `Terraform` 和 `Python` 构建 `Lambda` 函数，将 `Log Insights` 查询转换为 `CloudWatch` 指标。

[Executing a distributed shuffle without a MapReduce system](https://www.anyscale.com/blog/executing-a-distributed-shuffle-without-a-mapreduce-system)
A distributed shuffle is a data-intensive operation that usually calls for a system built specifically for that purpose. In this post, we’ll show how a distributed shuffle can be expressed in just a few lines of Python using Ray, a general-purpose framework whose core API contains no shuffle operations.

[A Low Poly Art Tutorial in Python](https://cosmiccoding.com.au/tutorials/lowpoly)

### 有趣的项目、工具和库

[CHIRP](https://github.com/cisagov/CHIRP)
一个用 `Python` 编写的查询收集工具。

[algobot](https://github.com/ZENALC/algobot)
具有图形用户界面的加密货币交易机器人。

[alkymi](https://github.com/MathiasStokholm/alkymi) 
`Alkymi` 是一个纯粹的 `Python (3.5+)` 库，用于描述和执行任务和管道，内置缓存和基于校验和的条件评估。

[qbr](https://github.com/kkoomen/qbr)
一个用 `Python 3` 和 `OpenCV` 编写的基于网络摄像头的3x3x3魔方求解器。

[Preql](https://github.com/erezsh/Preql)
一种解释的关系查询语言，可编译成 `SQL`。

[Codeline](https://github.com/synek/codeline) 
从你的源码注释中运行上下文感知的命令。

[gpt-neo](https://github.com/EleutherAI/gpt-neo)
使用 `mesh-tensorflow` 库，实现了类似于 `GPT2&GPT3` 的模型并行，能够扩展到完整的 `GPT3` 大小（甚至可能更多！）。

[Fickling](https://github.com/trailofbits/fickling)

[jut](https://github.com/kracekumar/jut)
一个在终端查看 `IPython/Jupyter` 笔记本的命令行工具。

[Episuite](https://github.com/perone/episuite)
在 `Python` 中的流行病学工具套件。
 
[Vinum](https://github.com/dmitrykoval/vinum)

[SmallPebble](https://github.com/sradc/SmallPebble)
